# Prompts Changelog

All notable changes to system prompts will be documented in this file.

## [1.0] - 2026-07-13

### Added
- **`triage_system.md` v1.0**: Initial implementation of the support ticket triage prompt. Includes explicit constraints for JSON output, schema definitions, and a robust prompt injection defense clause.
- **`account_summary_system.md` v1.0**: Initial implementation of the TAM account health summarizer prompt. Includes explicit instructions for extracting verbatim quotes from tickets, prompt injection defense, and Epistemic Safety safeguards.
